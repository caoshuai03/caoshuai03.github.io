# AcadHomepage

一个现代化且响应式的学术个人主页模板。

## 本地启动

1. 克隆仓库到本地：`git clone <repository-url>`
2. 安装 Jekyll 环境（需要 Ruby、RubyGems、GCC 和 Make），参考 [Jekyll 安装指南](https://jekyllrb.com/docs/installation/#requirements)
3. 安装依赖：`bundle install`
4. 启动服务器：`bash run_server.sh` 或 `bundle exec jekyll serve`
5. 在浏览器中访问：http://127.0.0.1:4000
